# Wordle Solver

> Solves [wordle](https://wordlegame.org) puzzles

Randomly selects words from the dictionary, and eliminates them based on the feedback from the game.
Not very intelligent.

## Use

Clone it, then install dependencies:
```shell
npm ci
```

Run with:
``` shell
npm run solve
```

## Development

### Run Locally

```shell
npm run solve
```

### Run Tests

```shell
npm test
```

### Build

```shell
npm run build
```
